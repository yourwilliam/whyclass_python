## mac安装opencv

```sh
$ brew install opencv

$ brew install cmake

$ pip install dlib --verbose
```

### windows 安装opencv

首先可以使用pip安装numpy

下载 [opencv](https://www.lfd.uci.edu/~gohlke/pythonlibs/#opencv)

拷贝到项目文件夹中，使用`(venv) D:\youyulab\opencv-code>pip install opencv_python-4.1.1-cp37-cp37m-win32.whl` 进行安装
